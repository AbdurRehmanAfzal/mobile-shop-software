DATA FLOW:
───────────────────────────────────────────────────────────────

User Action → State Update → Component Render → Visual Change

Example 1: Check IMEI 3
  User clicks checkbox
  → setShowImei3(true)
  → Component re-renders with showImei3 = true
  → {showImei3 && <BilingualInput...>} evaluates to true
  → IMEI 3 field appears in DOM
  → User can type value
  → Value stored in formData.imei3

Example 2: Uncheck IMEI 3
  User clicks checkbox
  → setShowImei3(false)
  → onInputChange('imei3', '') called
  → formData.imei3 = "" (cleared)
  → Component re-renders with showImei3 = false
  → {showImei3 && ...} evaluates to false
  → IMEI 3 field removed from DOM
  → Value completely cleared (no orphaned data)
